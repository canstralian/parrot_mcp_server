## Security

Security is a top priority for the Parrot MCP Server. We implement industry-standard security practices and cryptographically sign all releases.

### Release Verification

All official releases are signed with GPG keys. Before installing any release, verify its authenticity:

- **Quick Guide**: See [SECURITY.md](SECURITY.md#release-verification)
- **Detailed Instructions**: See [Release Verification Guide](docs/RELEASES.md)
- **Public Keys**: Available in the [gpg-keys/](gpg-keys/) directory

### Reporting Security Issues

If you discover a security vulnerability, please report it responsibly through our [Security Policy](SECURITY.md). We take all security reports seriously and respond promptly.
